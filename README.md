# 진로맵 Career Compass Pro

고등학생과 학부모가 대학·학과 선택을 직업, 기업, 연봉 수준, 생활 수준, 근무강도, 장단점, 리스크, 미래전망, 고등학교 준비 로드맵까지 연결해 보는 전문 진로 설계 웹앱입니다.

## 운영 원칙

- 모바일과 PC에서 모두 볼 수 있는 웹앱으로 만든다.
- 사용자의 MacBook이 꺼져 있어도 접속 가능해야 한다.
- 정적 배포는 GitHub Pages를 기본값으로 한다.
- 최신성이 중요한 입시·채용·연봉 정보는 앱이 단정하지 않고 공식 자료와 검색 링크로 확인하게 한다.

## 실행 방법

```bash
pnpm install
pnpm dev
```

정적 배포 빌드:

```bash
pnpm build
```

GitHub Pages 배포는 `.github/workflows/pages.yml`이 `main` 브랜치 push 때 자동으로 `out/` 결과물을 배포합니다.

## 주요 기능

- 홈 대시보드
- 5단계 학생 진로진단
- 학과맵 및 학과 상세
- 대학맵
- 직업맵 및 직업 상세
- 기업맵 및 기업 상세
- 비교실
- 메디컬 특화관
- 반도체 특화관
- 나의 진로 포트폴리오
- 학생·학부모·상담교사용 리포트
- 관리자 미리보기

## 데이터 구조

핵심 타입은 `src/types/career.ts`에 분리했습니다.

- `Major`: 학과, 과목, 성향, 관련 대학·직업·기업, 연봉, 생활수준, 장단점, 리스크
- `Job`: 직업, 진입경로, 자격/면허, 연봉, 생활수준, 미래전망
- `Company`: 기업, 산업, 채용 링크, OpenDART 검색, 관련 직무
- `University`, `UniversityMajor`: 대학과 모집단위 연결
- `ComparisonPreset`: 비교실 프리셋

초기 데이터는 `src/data/mockData.ts`에 있습니다.

## 향후 API 연동 계획

- 대학알리미: 학과, 취업률, 등록금, 장학금
- 대입정보포털 어디가: 모집단위, 전형, 입시 정보
- 커리어넷: 직업·학과 정보
- 워크넷/고용24: 채용정보
- OpenDART: 기업 공시
- YouTube Data API: 직무 인터뷰와 학과 소개 영상 큐레이션
- Supabase: 사용자 계정, 진단 결과, 포트폴리오 저장

## 연봉 데이터 주의사항

연봉 정보는 참고용 추정치입니다. 실제 연봉은 기업, 지역, 경력, 성과급, 근무형태, 개원·창업 여부에 따라 크게 달라질 수 있습니다. 평균 하나보다 경력별 범위, 소득 변동성, 워라밸, 건강, 가족시간, 번아웃 위험을 함께 봐야 합니다.

## 입시 데이터 주의사항

이 앱은 진로 탐색과 의사결정을 돕는 참고 도구입니다. 최종 입시 정보는 반드시 각 대학 입학처와 공식 모집요강을 확인해야 합니다. 모집단위명, 전형, 정원, 계약학과 조건, 장학·취업연계 조건은 매년 바뀔 수 있습니다.

## 미성년자 개인정보 보호

현재 버전은 브라우저 localStorage에만 포트폴리오를 저장합니다. 실서비스에서 사용자 계정을 도입할 경우 학생 개인정보, 상담 기록, 성향 진단 결과는 최소 수집, 보호자 동의, 접근권한 분리, 삭제 요청 절차를 포함해야 합니다.
